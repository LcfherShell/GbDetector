Usage Tutorial

  1. Make sure the credential.json file containing your API credentials is located in the current directory.
  2. Select a platform from the dropdown list (e.g., Instagram, Facebook, YouTube).
  3. Enter the ID of the media, post, or video you want to moderate.
  4. Click the Start Moderation button to begin the comment moderation process.
  5. The system log will display real-time progress and any detected issues.
  6. Click the Cancel button to stop the process at any time.
